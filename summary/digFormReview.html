Dig Form Review System

The Dig Form Review System is a custom workflow solution automating data collection and review. Field technicians are able to log information about pipeline digs and submit forms from a tablet.

Synopsis

Goals
  • Easy Form Submittals
  • Controlled approvals
  • Improve visibility
  • Decrease latency of information to the client
Process
  • Interviewed field techs to learn about forms and methods
  • Configured mobile form in Survey123
  • Constructed data workflow
  • Designed method for users to review submitted forms
Code
  • Survey123 - Configured form UI and data schema for form submittals
  • Angular - Built Angular component for review process
  • Angular - Integrated form submittal locations with web map
  • Python - Created Python scripts to migrate and calculate data

The previous system was disfunctional and inefficient, involving paper forms and emails. The basic idea was to provide a workflow allowing dig surveyors to submit forms from the field, with the ability for supervisors to review them.

For the Angular review component, our development team utilized the pair programming to.

Results

Overall this system was a great success. With some adjustments from user feedback along the way, we were able to achieve our goals and add value to the project.
